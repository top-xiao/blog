外測度の定義と諸性質

Definition 1.1 (区間の長さ). ℝ上の開区間Iと実数a < bに対し, 区間Iの長さ ℓ(I)を $$\ell (I) \coloneqq \begin{cases} 0, & I = \emptyset \\ b-a, & I = (a,b) \\ \infty, & I = (-\infty, a), (a, \infty), \mathbb{R} \end{cases}$$ と定める. |I|と略記することがある.

Definition 1.2 (Lebesgue外測度). ℝの部分集合Aに対し, Aの外測度λ*(A) を λ*(A) ≔ inf {∑n ∈ ℕ|In| ∣ A ⊆ ⋃n ∈ ℕIn,  ∀n ∈ ℕ; In がℝの開区間}

Proposition 1.3 (外測度の単調性). A, B ⊆ ℝに対し, A ⊆ Bとする. この時, λ*(A) ≤ λ*(B)が成立する. これを測度の単調性と呼ぶ.

Proof. Bを被覆する開区間の列が Aを被覆するので, 定義より明らかである. ◻

Proposition 1.4 (外測度の劣加法性). (Ak)k ∈ ℕ, ∀k ∈ ℕ, Ak ⊆ ℝとする. この時, λ*(⋃k ∈ ℕAk) ≤ ∑k ∈ ℕλ*(Ak) が成立する. これを外測度の劣加法性と呼ぶ.

Proof. ε > 0 任意のk ∈ ℕに対し, $A_k \subseteq \bigcup_{i \in \mathbb{N}}I_{k,i}, \sum_{i \in \mathbb{N}} |I_{k,i}| \leq \lambda^*(A_k) + \frac{\varepsilon}{2^k}$ を満たす開区間の列を取り, ⋃k ∈ ℕAk ⊆ ⋃k ∈ ℕ⋃i ∈ ℕIk, i が成立する. (可算集合の直積が可算集合より, 右辺は可算開区間の和になる). $$\lambda^*\left(\bigcup_{k \in \mathbb{N}}A_k \right) \leq \sum_{k \in \mathbb{N}} \sum_{i \in \mathbb{N}} |I_{k,i}| \leq \sum_{k \in \mathbb{N}} \lambda^*(A_k) + \sum_{k \in \mathbb{N}} \frac{1}{2^k} = \sum_{k \in \mathbb{N}}\lambda^*(A_k) + 2 \varepsilon$$

εの任意性より, λ*(⋃k ∈ ℕAk) ≤ ∑k ∈ ℕλ*(Ak). ◻

Proposition 1.5. ℝの一点集合の外測度が 0である.

Proof. 自然数kに対し, その点を含む長さが $\frac{1}{2^k}$ となるような列が取れる. その列について考えれば良い. ◻

Definition 1.6 (集合の平行移動). A ⊆ ℝ, t ∈ ℝに対し, t + A ≔ {t + a ∣ a ∈ A} と定める.

Proposition 1.7 (外測度の平行移動不変性). A ⊆ ℝ, t ∈ ℝに対し, λ*(A) = λ*(t+A) が成立する. これを外測度の平行移動不変性と呼ぶ.

Proof. 開区間の長さが平行移動の不変性を持つので, 定義通り確認するだけで済む. ◻

Proposition 1.8 (外測度と長さが適合する). a, b ∈ ℝ, a < bに対し, λ*([a,b]) = b − a.

Proof. λ*([a,b]) ≤ b − aが$I_k \coloneqq (a- \frac{1}{2^k}, b+ \frac{1}{2^k})$ の長さで評価すればすぐにわかる.

以下λ*([a,b]) ≥ b − aを示す. 帰納法を用いる. nに関する命題を以下のように定める

$$P_n \coloneqq \forall [a,b] \subseteq \mathbb{R}\text{に対し, $[a,b]\subseteq \bigcup_{k=1} ^n I_k$の時, $\sum_{k=1}^n |I_k| \geq b-a$ が成立する.}$$

[a,b]が一つの開区間Iで被覆される時, |I| ≤ b − a が明らかである. [a,b]が n + 1個の開区間で被覆される時, b ∈ In + 1 = (c,d), a < c < b < d としても一般性を失わない. この時, [a,c]が I1, …, Inによって被覆されるので, 帰納法の仮定が使えて,

$$\sum_{k=1}^{n+1} |I_k| = |I_{n+1}| + \sum_{k=1}^{n} |I_k| \geq b-c + c-a = b-a$$ 以上より, Pnが任意の nに対して成立する.

[a,b]を被覆する開区間の列に対し, [a,b]のコンパクト性より, 有限部分被覆が取れる. 上記の有限に議論に帰着される.

従って, λ*([a,b]) ≥ b − aが成立する. ◻

Proposition 1.9 (有限加法性の破綻). A, B ⊆ ℝ, A ∩ B = ∅で, λ*(A) + λ*(B) ≠ λ*(A∪B)となるような集合A, Bが存在する.

Proof. 背理法による.

群ℝ/ℚの[−1,1]における完全代表系 Vを選択公理を用いて一つ取り, q + V, q ∈ ℚ ∩ [−2,2]を取り, [−2,2] ⊆ ⋃q ∈ [−2,2] ∩ ℚ(q+V) と単調性と劣加法性より, λ*(V) > 0, ⋃q ∈ [−2,2] ∩ ℚ(q+V) ⊆ [−3,3]より, もし有限加法性が成立するならば, アルキメデスの原理に反する. ◻

Proposition 1.10. 以下の性質を満たす写像μが存在しない.

  1. μ : 𝒫(R) → [0,∞].

  2. 任意のℝ上の開区間Iに対してμ(I) = ℓ(I).

  3. 互いに素な可算集合列(Ak)k ∈ ℕ に対し, μ(⋃k ∈ ℕAk) = ∑k ∈ ℕμ(Ak).

  4. ∀A ∈ 𝒫(ℝ); ∀t ∈ ℝ, μ(t+A) = μ(A).

Proof. Proposition 1.9の証明を繰り返せばよい. ◻

σ-加法族と可測函数

ℝ上の外測度は長さを与える函数として集合の単調性, 劣加法性, 平行移動不変性と区間の長さと一致するなど素晴らしい性質を持っているが, 持ってほしい有限加法性は破綻してしまう. 私達はここで諦めてしまうのは早すぎる. ならば, 外測度という概念を捨てるか? という考えも受け入れ難い. ここでProposition 1.10の主張を見れば, すべての集合の長さを測ることを諦めるのが良さそうだろう.

しかし, 長さを測ることは写像を与えることであり, 写像の定義域をどのように定めればよいだろう? 定義域を持つべき性質は何だろう? 全体集合の長さを測りたい, 長さの持つ集合の差集合の長さを持ってほしい, 可算個の長さを持つ集合の和集合も長さを持ってくれると良い. これらの願望で生まれたのはσ-加法族である.

Definition 1.11 (σ-加法族と可測空間). 集合Xの部分集合族𝒮 ⊆ 𝒫(X) が以下の3条件を満たす時に, 𝒮 をσ-加法族という.

  1. X ∈ 𝒮.

  2. ∀A ∈ 𝒮; X \ A ∈ 𝒮.

  3. ∀(Ak)k ∈ ℕ ⊆ 𝒮; ⋃k ∈ ℕAk ∈ 𝒮.

この時(X,𝒮) を可測空間と呼ぶ. 𝒮 の元を𝒮-可測集合と呼ぶ, また単に可測集合と呼ぶこともある.

Definition 1.12 (Lebesgue可測集合族). 次のℝの部分集合族ℒ がLebesgue 可測集合族と呼ぶ. ℒ ≔ {E ∈ 𝒫(ℝ) ∣ ∀A ∈ 𝒫(ℝ); λ*(A∩E) + λ*(A\E) = λ*(A)} ℒ の元をLebesgue 可測集合と呼ぶ.

Lemma 1.13. A, B ∈ ℒ に対し, 以下の命題が成立する.

  1. A ∪ B ∈ ℒ.

  2. ℝ \ A ∈ ℒ.

  3. A ∩ B ∈ ℒ.

  4. A \ B ∈ ℒ.

Proof. M ∈ 𝒫(ℝ)を任意に取る. 劣加法性より, λ*(M∩(A∪B)) + λ*(M\(A\B)) ≥ λ*(M)が明らかで, 逆向きの不等号を示せば良い.

M ∩ (A∪B) = ((M∩A)⊔(M∩B)\A) より,

$$\label{eq:AuB} \begin{aligned} \lambda^*\left( M \cap (A\cup B) \right) &= \lambda^*\left( \left( M \cap A \right) \sqcup \left( M \cap B \right) \setminus A \right) \\ & \leq \lambda^*\left( M \cap A \right) + \lambda^*\left( (M \cap B)\setminus A \right) \\ & = \lambda^*\left( M \cap A \right) + \lambda^*\left( M \cap B \right) - \lambda^* \left( M \cap B \cap A \right). \end{aligned}$$

M \ (A∪B) = (M\B) \ Aより, $$\label{eq:minus-AuB} \begin{aligned} \lambda^*\left( M \setminus (A \cup B) \right) &= \lambda^* \left( \left( M \setminus B \right) \setminus A \right) \\ &= \lambda^*\left( M \setminus B \right) - \lambda^*\left( (M \setminus B ) \cap A \right) \\ &= \lambda^*\left( M \setminus B \right) - \lambda^*\left( (M \cap A)\setminus B \right) .\end{aligned}$$

また, λ*(M∩A) = λ*(M∩A∩B)) + λ*((M∩A)\B)より, 式[eq:AuB], [eq:minus-AuB]の両辺を足して, 次式が得られる.

$$\begin{aligned} \lambda^*\left( M \cap \left( A \cup B \right) \right) + \lambda^*\left( M \setminus \left( A\cup B \right) \right) & \leq \lambda^*(M) \end{aligned}$$

従って, A ∪ B ∈ ℒ が成立する.

ℝ \ A ∈ ℒ が明らかで, A ∩ B = ℝ \ ((ℝ\A)∪(ℝ\B)) より, A ∩ B ∈ ℒ が得られる. A \ B = A ∩ (ℝ\B) より, A \ B ∈ ℒ. ◻

Lemma 1.14. nを正の整数, A1, …, Anが互いに素なLebesgue 可測集合, X ∈ 𝒫(ℝ)とする. この時, $$\begin{aligned} \sum_{k=1}^{n} \lambda^*(X \cap A_k) \leq \lambda^*\left( X \cap \bigcup_{k=1}^n A_k \right) \end{aligned}$$ が成立する.

Proof. n = 1の時は明らかである. n = kの時に成立すると 仮定すると, n = k + 1の時, $$\begin{aligned} \sum_{k=1}^{n+1} \lambda^*\left( X \cap A_k \right) &= \sum_{k=1}^{n} \lambda^*\left(X \cap A_k \right) + \lambda^*\left(X \cap A_{n+1} \right) \\ &\leq \lambda^*\left(X \cap \bigcup_{k=1} ^n A_k \right) + \lambda^*\left( X \cap A_{n+1} \cap \bigcup_{k=1} ^{n+1}A_k \right) \\ &= \lambda^*\left(X \cap \bigcup_{k=1} ^n A_k \right) + \lambda^*\left(X \cap \bigcup_{k=1} ^{n+1}A_k \right) - \lambda^*\left( X \cap \bigcup_{k=1} ^n A_k \right) \\ &= \lambda^*\left( X \cap \bigcup_{k=1} ^{n+1}A_k \right) \end{aligned}$$ ◻

Theorem 1.15 (Lebesgue 可測集合族がσ-加法族). Lebesgue 可測集合族がσ-加法族である.

Proof. ℝ ∈ ℒ が明らかで, A ∈ ℒ に対し, ℝ \ A ∈ ℒ がLemma 1.13で既に示された. 以下可算個の可測集合列(Ak)k ∈ ℕ に対し, ⋃k ∈ ℕAk ∈ ℒ が成立することを示す. 簡単のため, 集合列が互いに素な前提で話を進める.

X ∈ 𝒫(ℝ) を任意に取る. $$\begin{aligned} \lambda^*\left(X \cap \bigcup_{k \in \mathbb{N}} A_k \right) + \lambda^*\left(X \setminus \bigcup_{k \in \mathbb{N}} A_k \right) = \lambda^*\left(X \right) \end{aligned}$$ を示せばよい, さらに劣加法性が成立するので, 左辺が右辺以下であることを示せば十分である. Bn = ⋃k ∈ ℕ ∧ k ≤ nAkと定める, 任意のn ∈ ℕに対し, $$\begin{aligned} \lambda^*\left(X \right) = \lambda^*\left(X \cap B_n \right)+ \lambda^*\left(X \setminus B_n \right) \end{aligned}$$ が成立する. この時, $$\begin{aligned} \lambda^*\left(X \right) & \geq \lambda^*\left(X \cap B_n \right) + \lambda^*\left(X \setminus \bigcup_{k \in \mathbb{N}} A_k \right) \tag*{(単調性)} \\ &\geq \sum_{k \in \mathbb{N} \land k \leq n} \lambda^*\left(A_k \right) + \lambda^*\left(X \setminus \bigcup_{k \in \mathbb{N}} A_k \right) \tag*{(Lemma \ref{lem:finite-additive})} \\ &\geq \sum_{k\in \mathbb{N} } \lambda^*\left( X \cap A_k \right) + \lambda^*\left(X \setminus \bigcup_{k \in \mathbb{N}} A_k \right) \tag*{($n \to \infty$)} \\ &\geq \lambda^*\left(X \cap \bigcup_{k \in \mathbb{N}} A_k \right) + \lambda^*\left(X \setminus \bigcup_{k \in \mathbb{N}} A_k \right) \tag*{(劣加法性)} \end{aligned}$$ 以上より, λ*(X) = λ*(X∩⋃k ∈ ℕAk) + λ*(X\⋃k ∈ ℕAk)が示された.

一般の場合, En ≔ An \ ⋃k ∈ ℕ ∧ k < nAk と定めれば, (Ek)k ∈ ℕ が互いに素なLebesgue 可測集合列で, ⋃k ∈ ℕAk = ⋃k ∈ ℕEkで, 先程の結果が適用される. ◻

Proposition 1.16 (任意の開集合がLebesgue可測である). ℝの任意の開集合Oに対し, O ∈ ℒ が成立する.

Proof. まず区間K ≔ [a, ∞), a ∈ ℝについて考えよう.

任意のX ∈ 𝒫(ℝ)に対し, 任意の正の実数ε に対し, ある開区間列(Ik)k ∈ ℕ が存在し, X ⊆ ⋃k ∈ ℕIkかつ∑k ∈ ℕ|Ik| ≤ λ*(X) + ε が成立する. J ≔ (−∞,a)と定める. 任意のk ∈ ℕに対し, Jk ≔ J ∩ Ik, Kk ≔ K ∩ Ikとする. X ∩ K ⊆ ⋃k ∈ ℕKk, X \ K ⊆ ⋃k ∈ ℕJk, 劣加法性より, $$\begin{aligned} \lambda^*\left(X \cap K \right) + \lambda^*\left(X \setminus K \right) \leq \sum_{k \in \mathbb{N}} |J_k| + |K_k| = \sum_{k\in \mathbb{N}} |I_k| \leq \lambda^*\left(X \right) + \varepsilon \end{aligned}$$ が成立する. εが任意なので, λ*(X∩K) + λ*(X\K) ≤ λ*(X). よって, [a, ∞) ∈ ℒが成立する. ℒがσ-加法族なので, $(a,b) = (\mathbb{R} \setminus [b , \infty) ) \cap \bigcup_{k \in \mathbb{N}}[a+\frac{1}{2^k} ,\infty)$ より, ∀a, b ∈ ℝ; (a,b) ∈ ℒが成立する. ℝは{(a,b) ∈ 𝒫(ℝ) ∣ a, b ∈ ℚ} で第2可算なので, ℝの任意の開集合Oに対し, O ∈ ℒが成立する. ◻

位相の話を思い出してください. 何らかの集合族が与えられた時, それを含む包含関係における最小の位相が存在する. その類似として, 集合 Xの部分集合族が与えられた時に, それを含む最小な σ-加法族が存在するか? という疑問に興味を持たなきゃいけないでしょう.

Proposition 1.17 (σ-加法族の生成). 任意の集合Xに対し, 以下のことが成立する.

  1. 𝒫(X)がX上 σ-加法族である.

  2. 任意のA ∈ 𝒫(X)に対し, σ(A) ≔ ⋂{𝒮 ⊆ 𝒫(X) ∣ A ⊆ 𝒮, Sがσ-加法族}がX上 σ-加法族である.

Proof. 簡単すぎるので省略. ◻

Definition 1.18 (最小なσ-加法族). XとA ⊆ Xに対し, σ(A) ≔ ⋂{𝒮 ⊆ 𝒫(X) ∣ A ⊆ 𝒮, 𝒮がX上の σ-加法族} をAの生成する最小の σ-加法族と呼ぶ.

Definition 1.19 (Borel 集合族). ℝの開集合系を𝒪ℝ とする. ℬ(ℝ) ≔ σ(𝒪ℝ) をℝのBorel集合族といい, ℬ(ℝ)の元をBorel集合と呼ぶ.

Proposition 1.20 (Borel集合がLebesgue可測集合). ℬ(ℝ) ⊆ ℒ が成立する.

Proof. 定義とProposition 1.16に従う. ◻

Definition 1.21 (拡張された実数). ℝ ∪ { ± ∞}上の順序と四則演算を次のように 拡張する. ∀a ∈ ℝに対して, $$\begin{aligned} - \infty <&a < \infty, \\ a+ \infty =\infty +a = \infty,&\ a+ (-\infty) = (-\infty) + a = (-\infty), \\ \infty + \infty = \infty,&\ (-\infty) +( -\infty) = (-\infty), \\ a \cdot \left( \pm \infty \right)& = \left( \pm \infty \right) \cdot a = \begin{cases} \pm \infty, &(a>0)\\ 0, & (a =0) \\ \mp \infty, & (a<0)\\ \end{cases}, \\ \infty \cdot \infty = \infty &\, (-\infty )\cdot (- \infty) = \infty, \\ \infty \cdot (-\infty) &\= (-\infty) \cdot (\infty) = (- \infty), \\ \frac{a}{\pm \infty} &= 0. \end{aligned}$$ これを$\overline{\mathbb{R}}$ と表記する.

Definition 1.22 (拡張されたBorel 集合). $\mathcal{\overline{\mathbb{R}}} \coloneqq \{B \in \mathcal{P}(\mathbb{R}) \mid B \cap \mathbb{R} \in \mathcal{B}(\mathbb{R})\}$ を拡張されたBorel 集合と呼ぶ. これが$\overline{\mathbb{R}}$ 上のσ-加法族であることは割と明らかである.

Definition 1.23 (可測函数). 可測空間(X,𝒮), (Y,𝒯)と写像f : X → Yに対し, ∀T ∈ 𝒯; f−1(T) ∈ 𝒮 を満たすならば, fを 𝒮-可測写像と呼ぶ. (Y,𝒯) = (ℝ,ℬ(ℝ)) の時, 単に可測函数と呼ぶこともある.

Remark 1.1. 定義を見ればわかると思うが, 可測性は定義域と終域両方に依存する性質で, (𝒮,𝒯)-可測函数と呼ぶのは合理的だけど, 終域は実数軸でBorel集合が装備される可測空間で考えるのは一般的なので, 終域の情報を捨てて, 簡略化した呼び方が上記のものになる.

Proposition 1.24 (可測の特徴づけ). (X,𝒮), (Y,𝒯)を可測空間, A ⊆ 𝒫(Y)に対し, σ(A) = 𝒯とする. 写像f : X → Yについて, 以下の2つの命題が同値である.

  1. ∀C ∈ A; f−1(C) ∈ 𝒮

  2. fが 𝒮-可測である.

Proof. ⟸が明らかで, ⟹を示す. ℳ ≔ {E ∈ 𝒯 ∣ f−1(E) ∈ 𝒮} とする. 逆像の性質より, ℳ がY上 σ-加法族になる. 最小性より, σ(A) ⊆ ℳで, 証明完了. ◻

Proposition 1.25 (可測函数の言い換え). 可測空間(X,𝒮) と写像f : X → ℝに対し, 以下の命題が同値である.

  1. fが 𝒮 可測である.

  2. ∀a ∈ ℝ; f−1((a,∞)) ∈ 𝒮.

  3. ∀a ∈ ℝ; f−1([a,∞)) ∈ 𝒮.

  4. ∀a ∈ ℝ; f−1((−∞,a)) ∈ 𝒮.

  5. ∀a ∈ ℝ; f−1((−∞,a]) ∈ 𝒮.

Proof. σ({(a,∞)∣a∈ℝ}) = σ({[a,∞)∣a∈ℝ}) = σ({(−∞,a)∣a∈ℝ}) = σ({(−∞,a]∣a∈ℝ}) = ℬ(ℝ)とProposition 1.24に従う. ◻

Definition 1.26 (可測空間の制限). 可測空間(X,𝒮) とE ⊆ Xに対し, $E \tilde{\cap} \mathcal{S} \coloneqq \{E \cap S \in \mathcal{P}(E)\mid S \in \mathcal{S}\}$ とする. これがE上 σ-加法族であり, 𝒮 のEへの制限と呼ぶ.

Proposition 1.27 (生成系の制限は本質的に一意である). 集合E ⊆ XとX上の集合族𝒜と可測空間(X,σ(𝒜))に対し, $E \tilde{\cap }\mathcal{A} \coloneqq \{E \cap C \in \mathcal{P}(E) \mid C \in \mathcal{A}\}$とする. この時, $\sigma(E \tilde{\cap} \mathcal{A}) = E \tilde{\cap} \sigma (\mathcal{A})$が成立する.

Proof. $\sigma (E \tilde{\cap} \mathcal{A}) \subseteq E \tilde{\cap} \sigma (\mathcal{A)}$ が成立することが生成されるσ-加法族の最小性に従う.

$\mathcal{M} \coloneqq \{B \in \mathcal{P}(X) \mid E \cap B \in \sigma (E\tilde{\cap} \mathcal{A})\}$ として, ℳ がX上 σ-加法族であり, 𝒜 ⊆ M より, $E \tilde{\cap} \sigma (\mathcal{A}) \subseteq \sigma(E \tilde{\cap} \mathcal{A})$ が得られる. ◻

Remark 1.2. σ-加法族の制限で登場する $\tilde{\cap}$が通常の意味での共通集合ではなくただの記号の濫用である.

Proposition 1.28 (連続函数が可測函数である). Borel集合X上で定義される函数 f : X → ℝが連続函数ならば, fは可測函数である. ただし, 可測空間を(X,X∩ℬ(ℝ))とする.

Proof. fが連続函数 ⇔ ∀O ∈ 𝒪ℝ; f−1(O) ∈ 𝒪X ∩ ℝ ⊆ X ∩ ℬ(ℝ). Proposition 1.24より証明が完了. ◻

Proposition 1.29 (単調函数が可測函数である). Borel集合Xで定義される単調函数fは可測函数である.

Proof. (a,∞)に対し, b ≔ inf f−1((a,∞))とすれば, f−1((a,∞)) = (b,∞) ∩ Xで, 可測性が確認された. ◻

Proposition 1.30 (可測函数の合成も可測函数). (X,𝒮) を可測空間, gをBorel 集合上で定義される可測函数, fが X 上の可測函数で, f の像がgの定義域に含まれるとする. この時, g ∘ fが 𝒮-可測である.

Proof. 位相空間論で同じタイプの問題をやっていたため, 省略. ◻

Proposition 1.31 (四則演算が可測である). (X,𝒮) を可測空間, f, g : X → ℝを可測函数とする. この時,

  1. f + g, f − g, fgが可測函数である.

  2. ∀x ∈ X; g(x) ≠ 0ならば, $\frac{f}{g}$も可測函数である.

Proof. $f-g = f+ (-g), fg =\frac{1}{2}\left( (f+g)^2 - f^2 - g^2\right), \frac{f}{g} = f \frac{1}{g}$ と連続函数が可測であることと可測函数の合成が可測であることより, f + gの可測性を示せば十分である.

a ∈ ℝを任意にとり, (f+g)−1((a,∞)) = ⋃r ∈ ℚ(f−1((r,∞))∩g−1(a−r,∞))) より, (f+g)−1((a,∞)) ∈ 𝒮 が成立する. ◻

これから函数列の収束に関連する議論に入るので, ここで$\overline{\mathbb{R}}$ の世界で, 可測性を議論する.

Proposition 1.32 ($\overline{\mathbb{R}}$ への可測函数). 可測空間(X,𝒮) と写像$f:X \to \overline{\mathbb{R}}$ に対し, ∀a ∈ ℝ; f−1((a,∞]) ∈ 𝒮 ならばfが 𝒮-可測である.

Proof. $\mathcal{B}(\overline{\mathbb{R}}) = \sigma \left( \{(a, \infty] \mid a \in \mathbb{R}\} \right)$ であることとProposition 1.24より直ちに従う. ◻

Remark 1.3. これより, 終域がℝの可測函数の終域を$\overline{\mathbb{R}}$ へ拡張しても可測であることが変わらない. これを以て, 可測函数の終域を特に明言しない限り, $\overline{\mathbb{R}}$ と思ってもよい. また今までℝで議論して得られた結論は$\overline{\mathbb{R}}$ でも成立する. 各自確認せよ.

Proposition 1.33 (可測函数の列から得られる可測函数). 可測空間(X,𝒮) と𝒮-可測函数列$f_k : X \to \overline{\mathbb{R}}, \ k \in \mathbb{N}$に対し, $g , h : X \to \overline{\mathbb{R}}$ を次のように定める: $$\begin{aligned} g(x)& \coloneqq \inf \{f_k(x) \mid k \in \mathbb{N}\}\\ h(x) &\coloneqq \sup \{f_k(x) \mid k \in \mathbb{N}\} .\end{aligned}$$ この時, g, hがどちらも 𝒮-可測である.

Proof. hについて, 任意のa ∈ ℝに対し, $$\begin{aligned} h^{-1}\left((a, \infty] \right) = \bigcup_{k \in \mathbb{N}}f_k^{-1}\left( (a, \infty] \right), \end{aligned}$$ が成立するので, hが可測函数であることが得られる.

また, $$\begin{aligned} g(x) = - \sup \{-f_k(x) \mid k \in \mathbb{N}\} \end{aligned}$$ が成立するから, gの可測性も得られる. ◻

Corollary 1.34 (可測函数列の各点収束先が可測函数). 可測空間(X,𝒮) 上で定義される可測函数列$f_k : X \to \overline{\mathbb{R}}$ がfに各点収束する時, fも可測函数になる.

Proof. gk ≔ sup {fi(x) ∣ i ≥ k, i ∈ ℕ} と定義する. 任意のk ∈ ℕに対し, gkがProposition 1.33 より可測である. f(x) = infk ∈ ℕgk(x) より, fの可測性が得られる. ◻

測度との初エンカ

Definition 1.35 (測度, 測度空間). (X,𝒮) を可測空間とする. 以下の2つの条件を満たす時, μ : 𝒮 → [0,∞] が(X,𝒮) 上の測度といい, (X,𝒮,μ) を測度空間と呼ぶ.

  1. μ(∅) = 0;

  2. 𝒮 上の任意の互いに素な集合列(Ek)k ∈ ℕ に対し, μ(⋃k ∈ ℕEk) = ∑k ∈ ℕμ(Ek).

二番目の条件を測度の完全加法性と呼ぶ.

Proposition 1.36 (単調性と差集合の測度). 測度空間(X,𝒮,μ)とD ⊆ Eを満たすD, E ∈ 𝒮に対し,

  1. μ(D) ≤ μ(E);

  2. μ(E\D) + μ(D) = μ(E)

Proof. E = D ∪ (E\D), D ∩ (E\D) = ∅に対し, 測度の完全加法性を適用すれば良い. ◻

Proposition 1.37 (測度の劣加法性). 測度空間(X,𝒮,μ) と可測集合列(Ek)k ∈ ℕに対し, $$\begin{aligned} \mu \left( \bigcup_{k \in \mathbb{N}} E_k \right) \leq \sum_{k\in \mathbb{N}} \mu \left( E_k \right) \end{aligned}$$ が成立する.

Proof. Fmin ℕ ≔ ∅, Fk + 1 ≔ ⋃i ∈ ℕ ∧ i ≤ kEi とする. この時, (Ek\Fk)k ∈ ℕ が互いに素な可測集合列で, $$\begin{aligned} \mu \left( \bigcup_{k \in \mathbb{N}} E_k \right) &= \mu \left( \bigcup_{k \in \mathbb{N}}(E_k \setminus F_k ) \right) \tag{集合の一致} \\ &= \sum_{k\in \mathbb{N}}\mu (E_k \setminus F_k ) \tag{完全加法性}\\ &\leq \sum_{k\in \mathbb{N}} \mu (E_k) \tag{単調性} . \end{aligned}$$ ◻

Proposition 1.38 (測度の連続性). 測度空間(X,𝒮,μ) と包含関係について単調増大な可測集合の鎖(Ek)k ∈ ℕ に対し, $$\begin{aligned} \mu\left( \bigcup_{k\in \mathbb{N}} E_k \right) = \lim_{k \to \infty} \mu (E_k) \end{aligned}$$ が成立する.

Proof. Emin ℕ − 1 = ∅として, 単調増大性より, $$\begin{aligned} \bigcup_{k=\min \mathbb{N}}^{\infty} E_k = \bigcup_{k = \min \mathbb{N}}^{\infty}\left( E_k \setminus E_{k-1} \right) \end{aligned}$$ が成立し, 右辺は互いに素な集合の直和になる. これについて完全加法性を適用すると

$$\begin{aligned} \mu \left( \bigcup_{k \in \mathbb{N}} E_k \right) &= \lim_{k \to \infty} \sum_{n= \min \mathbb{N}}^{k} \mu \left( E_n \setminus E_{n-1}\right) \\ &= \lim_{k\to \infty} \sum_{n=\min \mathbb{N}}^{k} \mu \left( E_n \right) - \mu \left( E_{n-1} \right) \\ &= \lim_{k \to \infty} \mu \left( E_k \right) \end{aligned}$$ ◻

Proposition 1.39 (減少列の測度). 測度空間(X,𝒮,μ) と包含関係について単調減少な可測集合の鎖(Ek)k ∈ ℕ に対し, ∃N ∈ ℕ; μ(EN) < ∞ならば, $$\begin{aligned} \mu \left( \bigcap_{k \in \mathbb{N}}E_k \right) = \lim_{k \to \infty} \mu (E_k) \end{aligned}$$ が成立する.

Proof. k ∈ ℕに対し, Fk ≔ EN \ Fkと定める. この時, (Fk)k ∈ ℕ が集合の包含関係について単調増大列になる. Proposition 1.38を適用すると, $$\begin{aligned} \mu \left( E_N \setminus \bigcap_{k \in \mathbb{N}}E_k \right) = \lim_{k \to \infty} \mu \left( E_N \setminus E_k \right) \end{aligned}$$

⋂k ∈ ℕEk ⊆ ENより, μ(⋂k ∈ ℕEk) ≤ μ(EN) < ∞. 上式から, $$\begin{aligned} \mu (E_N) - \mu \left( \bigcap_{k\in \mathbb{N}} E_k \right) = \lim_{k \to \infty, \land k \geq N } \mu (E_N) - \mu (E_k) \end{aligned}$$ が得られ, 両辺からμ(EN)を引いて  − 1を掛けると, $$\begin{aligned} \mu \left( \bigcap_{k \in \mathbb{N}} E_k \right) = \lim_{k \to \infty} \mu (E_k) \end{aligned}$$ が成立する. ◻

Proposition 1.40. 測度空間(X,𝒮,μ) とD, E ∈ 𝒮 に対し, $$\begin{aligned} \mu \left( D \cup E \right) + \mu \left( D \cap E \right) = \mu (D) + \mu (E) \end{aligned}$$ が成立する.

Proof. $$\begin{aligned} \mu \left( D \cup E \right) + \mu \left( D \cap E \right) &= \mu \left( (D \setminus E) \sqcup (E \setminus D) \sqcup \left( E \cap D \right) \right) + \mu \left( D \cap E \right) \\ &= \mu \left( D \setminus E \right) + \mu \left( D \cap E \right) + \mu \left( E \setminus D \right) + \mu \left( D \cap E \right) \\ &= \mu(D) + \mu (E). \end{aligned}$$ ◻

Definition 1.41 (零集合). 測度空間(X,𝒮,μ) とする. $$\begin{aligned} \mathcal{N}_X \coloneqq \{A \subseteq X \mid \exists B \in \mathcal{S}; A \subseteq B \land \mu (B) = 0\} \end{aligned}$$ が零集合族といい, 𝒩X の元を零集合と呼ぶ.

Definition 1.42 (測度の完備化). 測度空間(X,𝒮,μ) とする. $$\begin{aligned} \overline{\mathcal{S}} \coloneqq \{M \cup N \mid M \in \mathcal{S}\land N \in \mathcal{N}_X\} \end{aligned}$$ とおき, 写像$\overline{\mu} : \overline{\mathcal{S}} \to [0,\infty]$ を M ∈ 𝒮; N ∈ 𝒩X に対し, $$\begin{aligned} \overline{\mu}(M \cup N) \coloneqq \mu (M) \end{aligned}$$ と定める. $\left( X , \overline{\mathcal{S}}, \overline{\mu} \right)$ が(X,𝒮,μ) の完備化という.

Proposition 1.43 (完備化のwell-defined性). 測度空間(X,𝒮,μ) の完備化$\left( X, \overline{\mathcal{S}}, \overline{\mu} \right)$ に対し, 以下のことが成立する.

  1. $\overline{\mathcal{S}}$ がσ-加法族である.

  2. $\overline{\mu}$ がwell-definedである.

  3. $\forall E \in \mathcal{S}; \overline{\mu}(E) = \mu(E)$, これにより, $\left( X, \overline{\mathcal{S}}, \overline{\left( \mu \right) } \right)$ が測度空間になる.

  4. $\forall M \in \overline{\mathcal{S}}; \forall N \in \mathcal{P}(X); \left( \mu (M) = 0 \land N \subseteq M \implies N \in \overline{\mathcal{S}} \right)$.

  5. S̃ ≔ {A ⊆ X ∣ ∃D, E ∈ 𝒮; E ⊆ A ⊆ D ∧ μ(D\E) = 0} とすると, $\tilde{S} = \overline{S}$ が成立する.

Proof. 1. X = X ∪ ∅ より, $X \in \overline{\mathcal{S}}$ が成立する. M, O ∈ 𝒮, N ∈ 𝒩X, μ(O) = 0, N ⊆ O に対し, X \ (M∪N) = (X\(M∪O)) ∪ (O\(M∪N)) が成立するので, $X \setminus \left( M \cup N \right) \in \overline{\mathcal{S}}$ が成立する. (Mk,Nk,Ok)k ∈ ℕ; ∀k ∈ ℕ; (Mk,Ok∈𝒮∧Nk⊆Ok∧μ(Ok)=0) とする. ⋃k ∈ ℕ(Mk∪Nk) = ⋃k ∈ ℕMk ∪ ⋃k ∈ ℕNk, μ(⋃k ∈ ℕOk) = 0 より, 完全加法性が成立する. よって, $\overline{\mathcal{S}}$ がσ-加法族である.

2. M1, M2, O1, O2 ∈ 𝒮, N1 ⊆ O1, N2 ⊆ O2, μ(O1) = μ(O2) = 0, M1 ∪ N1 = M2 ∪ N2とする. $\overline{\mu}(M_1 \cup N_1) = \mu (M_1) = \mu (M_1 \cup O_1 \cup O_2) = \mu \left( M_2 \cup O_1 \cup O_2 \right) = \mu (M_2) = \overline{\mu}\left( M_2\cup N_2 \right)$. したがって, $\overline{\mu}$がwell-definedである.

3. E ∈ 𝒮に対し, $\overline{\mu}(E)= \overline{\mu}(E \cup \emptyset) = \mu (E)$. このあとの議論はroutineなので省略.

4. $\forall M \in \overline{\mathcal{S}}; \forall N \in \mathcal{P}(X); \left(\mu (M) = 0 \land N \subseteq M\right)$に対し, E, F ∈ 𝒮, G ∈ 𝒩X, M = E ∪ G, G ⊆ F, μ(F) = 0 を満たすE, F, Gを取り, N ⊆ E ∪ F, μ(E∪F) = 0より, N ∈ 𝒩X, N = ∅ ∪ N より, $N \in \overline{\mathcal{S}}$ が成立する.

5. D, E ∈ 𝒮, E ⊆ A ⊆ D, μ(D\E) = 0 とすると, A = E ∪ (D\E) で, $A \in \overline{\mathcal{S}}$ が成立し, $\tilde{\mathcal{S}} \subseteq \overline{\mathcal{S}}$ が得られた. 一方, M, O ∈ 𝒮; N ⊆ O; μ(O) = 0 に対し, M \ O ⊆ M ∪ N ⊆ M ∪ O, μ((M∪O)\(M\O)) = μ(O) = 0より, $M\cup N \in \tilde{\mathcal{S}}$ が成立し, $\overline{\mathcal{S}} \subseteq \tilde{\mathcal{S}}$ が得られる. ◻

Definition 1.44 (Gδ, Fσ 集合). 位相空間(X,𝒪X) に対し, 可算個の開集合の共通部分で表せるものをGδ 集合, 可算個の閉集合の和集合で表せるものをFσ 集合と呼ぶ.

Proposition 1.45 (Lebesgue可測集合のBorel集合による近似). 任意のLebesgue可測集合A ∈ ℒに対し, 以下のことが成立する:

  1. 任意のε > 0に対し, ある開集合 B が存在し, A ⊆ B ∧ λ * (B\A) < ε.

  2. 任意のε > 0に対し, ある閉集合Bが存在し, B ⊆ A ∧ λ*(A\B) < ε.

  3. あるGδ 集合Gが存在し, A ⊆ G ∧ λ*(G\A) = 0.

  4. あるFσ 集合Fが存在し, F ⊆ A ∧ λ*(A\F) = 0.

Proof. 1. λ*(A) < ∞の時, あるAを被覆する開区間の列(Ik)k ∈ ℕ が存在し, ∑k ∈ ℕ|Ik| < λ*(A) + εが成立する. この時, B ≔ ⋃k ∈ ℕIkと定めると, A ∈ ℒ より λ*(B\A) = λ*(B) − λ*(A) ≤ ∑k ∈ ℕ|Ik| − λ*(A) < ε が得られる.

λ*(A) = ∞ の時, k ∈ ℤに対し, $E_k \coloneqq [k, k+1) , \ A_k \coloneqq A \cap E_k, \ \varepsilon_k = \frac{\varepsilon}{3 \cdot 2^{|k|}}$と定める. 各k ∈ ℤに対し, 前半の結論を適用すると, 開集合Bk ⊇ Akが存在し, λ*(Bk\Ak) < εkが成立する. B ≔ ⋃k ∈ ℤBkとすれば, λ*(B\A) ≤ λ*(⋃k ∈ ℤ(Bk\Ak)) ≤ ∑k ∈ ℤλ*(Bk\Ak) < ∑k ∈ ℤεk = ε.

2. A ∈ ℒ より, ℝ \ A ∈ ℒ が成立する. ℝ \ Aに対し, 1. の結論を適用すると開集合ℝ \ Bが存在し, ℝ \ A ⊆ ℝ \ B ∧ λ*((ℝ\B)\(ℝ\A)) < εが成立する. この時, 集合Bが望む閉集合である.

3. 各自然数kに対し, $\varepsilon_k \coloneqq \frac{1}{2^k}$ と定めて, 1. を適用して, A ⊆ Bk ∧ λ*(Bk\Ak) < εk を満たす開集合Bkを取り, B ≔ ⋂k ∈ ℕBkとすれば良い.

4. 各自然数kに対し, $\varepsilon_k \coloneqq \frac{1}{2^k}$ とおく, 2.を適用して, Bk ⊆ A ∧ λ*(A\Bk) < εkを満たす閉集合Bkを取り, B ≔ ⋃k ∈ ℕBkとすれば良い. ◻

おまけ

この節では, 今までの議論を振り返り, Lebesgue測度という概念を拡張することを目指す. Lebesgue測度で出てきたσ-加法族に関連する議論は, 外測度の基本性質しか使われていないと いうことに気をつけてください. それは何を表しているかというと, 完全加法性を満たす σ-加法族を考えて測度を作り出す前に, 外測度を先に構成して, それで自ずから測度が 生まれてくるわけ. ここで, 定義と主張だけ述べておいて, 証明は今までの議論をそのままコピーすれば できるのだ.

Definition 1.49 (外測度). Xを集合, 集合 X上の外測度 μ* : 𝒫(X) → [0,∞]は以下の3つの性質を満たす写像である. (ドヤ顔をしているのはわかるよw)

  1. μ*(∅) = 0,

  2. A ⊆ B ⊆ Xならば, μ*(A) ≤ μ*(B).

積分

今章から測度に基づく積分論を展開する. 流れとしては単函数と呼ばれるものを用いて, 可測函数を近似し, 単函数の積分の極限をもって可測函数の積分を与える単調収束定理とそれから発生するFatouの補題とLebesgueの優収束定理を紹介する予定である.

積分の定義と簡単な性質

Definition 2.1 (単函数). 可測空間(X,𝒮) 上で定義される有限個の値しか取らない可測函数f : X → ℝ を単函数と呼ぶ. $$\begin{aligned} \mathcal{E}\coloneqq \{f:X \to \mathbb{R} \mid f \text{が単函数である}\} \end{aligned}$$ これを単函数全体の集合と呼び, $$\begin{aligned} \mathcal{E}^{+}\coloneqq \{f \in \mathcal{E} \mid \forall x \in X; f(x) \geq 0\} \end{aligned}$$ を非負単函数全体の集合と呼ぶ.

Definition 2.2 (非負単函数の積分). 測度空間(X,𝒮,μ) 上で定義される非負単函数f : X → [0, ∞) に対し, ある(Ek)k = 1nが𝒮-可測集合列と非負実数列(ck)k = 1nが存在し, $$\begin{aligned} f(x) = \sum_{k=1}^{n} c_k \mathds{1}_{E_k}(x) \end{aligned}$$ の時, $$\begin{aligned} I_{\mu }(f) \coloneqq \sum_{k=1}^{n} c_k \mu(E_k) \end{aligned}$$

を単函数の積分と呼ぶ.

Proposition 2.3 (単函数の表示変換). 測度空間(X,𝒮,μ)上のあるa1, a2, …, am ∈ [0, ∞)と𝒮-可測集合列A1, A2, …, Amに対し, あるc1, c2, …, cn ∈ [0, ∞)と互いに素な可測集合列$C_1,C_2,\ldots,C_n, \bigcup_{i =1}^n C_i = \bigcup_{i = 1}^m A_i$を満たすものが存在し, $$\begin{aligned} \sum_{k=1}^{m} a_k \mathds{1}_{A_k}(x) = \sum_{k=1}^{n} c_k \mathds{1}_{C_k}(x) \\ \sum_{k=1}^{m} a_k \mu(A_k) = \sum_{k=1}^{n} c_k \mu (C_k) \end{aligned}$$ を満たす.

Proof. a2, A2について, $$\begin{aligned} a_1\mathds{1}_{A_1}(x) + a_2\mathds{1}_{A_2}(x) = a_1 \mathds{1}_{a_1 \setminus A_2} (x) + a_2 \mathds{1}_{A_2 \setminus A_1}(x) + (a_1+a_2) \mathds{1}_{A_1\cap A_2}(x)\\ a_1\mu(A_1) +a_2\mu(A_2) = a_1 \mu (A_1 \setminus A_2)+ a_2\mu (A_2 \setminus A_1) +(a_1+a_2) \mu(A_1 \cap A_2) \end{aligned}$$ が成立し, 右辺の3つの互いに素な集合の測度の一次結合をb1μ(B1) + b2μ(B2) + b3μ(B3)として, これらに, a3, A3との対称差を考えて新しい互いに素な集合の測度の一次結合を作り, 有限回繰り返すと, am, Amによる対称差での列が望む結果である. ◻

Proposition 2.4 (単函数の積分がwell-defined). 測度空間(X,𝒮,μ)とa1, a2, …, am, b1, b2, …, bn ∈ [0,∞] と𝒮-可測集合A1, A2, …, Am, B1, B2, …, Bnに対し, $\sum_{k=1}^{m} a_k \mathds{1}_{A_k} = \sum_{k=1}^{n} b_k\mathds{1}_{B_k}$ ならば, $$\begin{aligned} \sum_{k=1}^{m} a_{k} \mu (A_k) = \sum_{k=1}^{n} b_k \mu (B_k). \end{aligned}$$

Proof. Proposition 2.3より, A1, …, Amが互いに素, B1, …, Bnが互いに素であると仮定してもよい. この時, $$\begin{aligned} \sum_{k=1}^{m} a_k \mu(A_k) = \sum_{k=1}^{m} \sum_{l =1}^{n} a_k \mu (A_k \cap B_l) \\ \sum_{l = 1}^{n} b_l \mu (B_l) = \sum_{l =1}^n\sum_{k=1}^{m} b_l \mu (B_l \cap A_k) \end{aligned}$$

単函数が一致するので, ∀1 ≤ k ≤ m, 1 ≤ l ≤ nに対し, akμ(Ak∩Bl) = blμ(Bl∩Ak) が成立する. よって積分値が一致する. ◻

Proposition 2.5 (非負単関数の積分の性質). 測度空間(X,𝒮,μ) について,

  1. A ∈ 𝒮, Iμ(𝟙A) = μ(A).

  2. f + g ∈ ℰ+, Iμ(f+g) = Iμ(f) + Iμ(g)

  3. ∀a ≥ 0; ∀f ∈ ℰ+; Iμ(af) = aIμ(f).

  4. f, g ∈ ℰ+; (f≥g⟹Iμ(f)≥Iμ(g)).

Proof. 1,2,3は定義を確認するで明らか, 4はIμ(f) − Iμ(g) を定義通りに展開して非負単函数f − gの積分になるから 0以上になる. ◻

Definition 2.6 (非負函数の積分). (X,𝒮,μ) 上で定義される非負値函数f : X → [0,∞]に対し, $$\begin{aligned} \int_X f d\mu \coloneqq \sup \{I_{\mu}(g) \mid g \in \mathcal{E}^{+}; \forall x \in X; g(x) \leq f(x)\} \end{aligned}$$ を函数fの μに関する積分と呼ぶ. ∫Xf(x)dμ(x)と表すことがある.

Definition 2.7. 函数$f,g:X \to \overline{\mathbb{R}}$ に対し, $$\begin{aligned} f \leq g\overset{\text{def}} \iff \forall x \in X; f(x) \leq g(x) \\ f < g \overset{\text{def}} \iff \forall x \in X; f(x) < g(x) \\ f \geq g \overset{\text{def}} \iff \forall x \in X; f(x) \geq g(x) \\ f > g \overset{\text{def}} \iff \forall x \in X; f(x) > g(x) \end{aligned}$$ とする.

Proposition 2.8 (非負函数の積分の基本性質). 測度空間(X,𝒮,μ) 上の非負値函数f, gについて,

  2. ∀x ∈ X; f(x) ≥ g(x)ならば, ∫Xfdμ ≥ ∫Xgdμ.

Theorem 2.9 (単調収束定理). 測度空間(X,𝒮,μ)上の非負単調可測函数列(fk)k ∈ ℕ に対し, ∀x ∈ X; f(x) ≔ limk → ∞fk(x) と定めると, $$\begin{aligned} \lim_{k \to \infty} \int_X f_k d \mu = \int_X f d\mu \end{aligned}$$

が成立する.

Proof. limk → ∞∫Xfkdμ ≤ ∫Xfdμが成立することが積分が函数の大小関係を保つことから得られる.

limk → ∞∫Xfk ≥ ∫Xfdμを示せば十分. 以下, g ∈ ℰ+; g ≤ f を満たすgを任意に取り, limk → ∞∫Xfkdμ ≥ Iμ(g) を示す. 互いに素な可測集合列と非負実数列が存在し, $g = \sum_{k=1}^{n} a_k \mathds{1}_{A_k}$ とする. t ∈ (0,1)を任意に取り, $$\begin{aligned} E_k = \{ x \in X \mid f_k(x) \geq t \sum_{k=1}^{n} a_k \mathds{1}_{A_k}(x)\} \end{aligned}$$ とする. Proposition 2.19. 測度空間(X,𝒮,μ)上の2つの可測函数$f,g : \to \overline{\mathbb{R}}$に対し, N ∈ 𝒮, μ(N) = 0 とする. この時,

が得られる. ◻

Theorem 2.20 (Lebesgue 優収束定理). 測度空間(X,𝒮,μ) 上の可測函数の列(fk)k ∈ ℕ, $f_k: X \to \overline{\mathbb{R}}$に対し, 以下の2条件を満たすとする.

  1. (fk)k ∈ ℕ がfに各点収束する.

  2. ある非負可測函数 g : X → [0,∞]が存在し, supk ∈ ℕ|fk| ≤ g, ∫Xgdμ < ∞ が成立する.

この時, fも可積分で, $$\begin{aligned} \lim_{k \to \infty} \int_X f_k d\mu = \int_X f d\mu \end{aligned}$$ が成立.

Proof. 可積分性はsupk ∈ ℕ|fk| ≤ g に従う. 同様に, 各fkも可積分. 以下, ∀k ∈ ℕ; ∀x ∈ X; |fk(x)| < ∞, |f(x)| < ∞と仮定する. g + f ≥ 0, g − f ≥ 0となる. Fatou’s lemma より, $$\begin{aligned} \int_X g \pm f d\mu = \int_X \liminf_{k \to \infty}(g \pm f_k) d\mu \leq \liminf_{k \to \infty} \int_X g \pm f_k d\mu \end{aligned}$$

+の部分では, $$\begin{aligned} \int_X g d\mu + \int_X f d\mu \leq \int_X g d\mu + \liminf_{k \to \infty} \int_X f_k d\mu \end{aligned}$$

−の部分では $$\begin{aligned} - \int_X f d\mu \leq \liminf_{k\to \infty}\int_X -f_k d\mu = - \limsup_{k \to \infty}\int_X f_k d\mu \end{aligned}$$

以上より, $$\begin{aligned} \int_X f d\mu \leq \liminf_{k \to \infty} \int_X f_k d\mu \leq \limsup_{k \to \infty} \int_X f_k d\mu \leq \int_X f d\mu \end{aligned}$$ が得られ, $$\begin{aligned} \liminf_{k \to \infty } \int_X f_k d\mu = \limsup_{k \to \infty} \int_X f_k d\mu = \lim_{k \to \infty} \int_X f_k d\mu = \int_X f d\mu \end{aligned}$$となる.

各fkと fの可積分性より, μ({|fk|=∞}) = μ({|f|=∞}) = 0その集合を除いたところで上記の議論をすれば良い. ◻

Theorem 2.26. fを有界閉区間 [a,b]上の有界函数, fは [a,b]上Riemann 可積分とする. この時,

Lp空間の定義と基本性質

Definition 3.1. p ∈ [1, ∞)とする. $$\begin{aligned} \mathscr{L}^p(\mu ) \coloneqq\left\{f:X \to \mathbb{R}; \text{$f$は $\mathcal{S}$-可測で$\int_X |f|^p d \mu < \infty$}\right\} \end{aligned}$$ f − g = 0, μ-a.e.を満たすf, g ∈ ℒp(μ)をf ∼ gとする. ∼が同値関係であり, Lp(μ) ≔ ℒ(μ)/∼ をLp-空間と呼ぶ. また, f ∈ Lp(μ)に対し, (ここでfを代表元として取る) $$\begin{aligned} \|f\|_p \coloneqq \left( \int_X |f|^p d \mu \right) ^{\frac{1}{p}} \end{aligned}$$ をfの Lp-ノルムと呼ぶ. (函数としてwell-definedである)

Proposition 3.2. Lp(μ)はℝ線形空間である.

Proof. ∀α, β ≤ 0に対して, xpの凸性より, (α+β)p ≤ 2p − 1(αp+βp) が成立する. これを利用して, $$\begin{aligned} \int_X |f+g|^p d \mu \leq 2^{p-1}\left( \int_X |f|^p d\mu + \int_X |g|^p d \mu \right) < \infty \end{aligned}$$ 定数倍に関する議論は明らかである. ◻

Lemma 3.3. p ∈ (1,∞)とし, q ∈ (1,∞)を$\frac{1}{p}+\frac{1}{q}=1$ となるように取る. qを pの共役指数と呼ぶ. この時, ∀α, β ≥ 0に対し, $$\begin{aligned} \alpha \beta \leq \frac{\alpha ^p}{p}+ \frac{\beta ^q}{q} \end{aligned}$$ が成立する. 等号成立は ⇔ β = αp − 1 である.

Proof. a > 0, b > 0とする. この時, ln の凸性と単調性より$\ln(ab) \leq \ln\left(\frac{1}{p}\alpha ^{p} + \frac{1}{q}\beta ^q\right)$ を示せば良い. $$\ln \left(\frac{1}{p}\alpha ^p + \frac{1}{q}\beta ^{q}\right) \geq \frac{1}{p}\ln (\alpha ^p) + \frac{1}{q}\ln (\beta ^q) = \ln (\alpha \beta ).$$ ◻

Theorem 3.4. p, q ∈ (1,∞)を互いに共役指数とする. f ∈ Lp(μ), g ∈ Lq(μ)に対し, fg ∈ L1(μ)が成立し, かつ $$\begin{aligned} \int_X |fg| d \mu \leq \|f\|_p \|g\|_q \end{aligned}$$ が成立する. 等号の成立条件 ⇔ ∥g∥qq|f(x)|p = ∥f∥pp|g(x)|q, μ-a.e. で成立する.

Proof. ∥f∥p, ∥g∥q > 0とする. $$\begin{aligned} \int_X \frac{|f|}{\|f\|_p} \frac{|g|}{\|g\|_q} d \mu \leq \int_X \frac{1}{p}\left(\frac{|f|}{\|f\|_p}\right)^{p} + \frac{1}{q}\left(\frac{|g|}{\|g\|_q}\right)^{q} d \mu = \frac{1}{p}\frac{\|f\|_p^p}{\|f\|_p^p} + \frac{1}{q} \frac{\|g\|_q^q}{\|g\|_q^q} =1 \end{aligned}$$ よって, $$\begin{aligned} \|fg\|_1 \leq \|f\|_p \|g\|_q \end{aligned}$$ が成立する. ◻

Theorem 3.5. p ∈ [1, ∞)とし, f, g ∈ Lp(μ) とする. この時, $$\begin{aligned} \|f+g\|_p \leq \|f\|_p + \|g\|_p \end{aligned}$$ が成立する.

Proof. p > 1とする. $q = \frac{p}{p-1}$ として, Theorem 3.4の不等式を適用すると, $$\begin{aligned} \|f+g\|_p^p &= \int_X |f+g| |f+g|^{p-1} d \mu \leq \int_X |f| |f+g|^{p-1} d \mu + \int_X |g| |f+g|^{p-1} d \mu \\ &\leq \|f\|_p \| \left( |f + g| \right)^{p-1} \|_{q} + \|g\|_p \| \left( |f+g| \right) ^{p-1}\|_q \\ & \leq \left( \|f\|_p + \|g\|_p \right) \|f+g\|_p^{p-1} \end{aligned}$$ これより, ∥f + g∥p ≤ ∥f∥p + ∥g∥pが得られる. ◻

Proposition 3.8. Lp(μ)が完備である.

Theorem 3.11. 概収束かつLp-norm収束 ⟹ Lp-収束.

すなわち, f, fk ∈ Lp(μ), k = 1, 2…がfk → f, μ-a.e. かつ limn → ∞∥fn∥p = ∥f∥pを満たせば, $$\begin{aligned} \lim_{n \to \infty} \|f_n -f\|_p = 0 \end{aligned}$$ が成立する.

Proof. k = 1, 2, …に対し, $$\begin{aligned} g_k = 2^{p-1}\left( |f_k|^p + |f|^p \right) - |f_k-f|^p \end{aligned}$$ と置くと, gk ≥ 0, かつ gk → 2p|f| μ-a.e. が成立する. よって, Fatouの補題より, $$\begin{aligned} 2^p \int_X |f|^p d \mu &= \int_X \liminf_{k \to \infty}g_k d \mu \leq \liminf_{k\to \infty} \int_X g_k d \mu \\ &= \liminf_{k\to \infty} \left(2^p \left( \int_X |f_k|^p d \mu + \int_X |f|^p d \mu \right) - \int_X |f_k -f|^p d \mu \right) \\ &= 2^p \int_X |f|^p \mu - \limsup_{k\to \infty}\int_X |f_k-f|^p d \mu \end{aligned}$$ 従って, limk → ∞∥fk − f∥p = 0. ◻

Corollary 3.12. ℰ ∩ Lp(μ)はLp(μ)において稠密である. つまり, ∀f ∈ Lp(μ), ∃{fk}k = 1∞ ⊆ ℰ ∩ Lp(μ) s.t. $$\begin{aligned} \lim_{k \to \infty} \|f_k -f\|_p = 0 \end{aligned}$$ が成立する.

Proof. fが非負の時, {fk}k = 1∞ ⊆ ℰ+で, fに単調的に各点収束するものが取れる. すると, 0 ≤ fk ≤ f から, $$\begin{aligned} \int_X |f_k|^p d\mu \leq \int_X |f|^p d \mu < \infty, \end{aligned}$$ よって, fk ∈ Lp(μ)が得られ, 単調収束定理より, $$\begin{aligned} \lim_{k \to \infty} \|f_k\|_p^p = \int_{X}|f|^p d \mu = \|f\|_p^p \end{aligned}$$ よって, Lp-norm 収束が言えた. Theorem 3.11より, 命題が従う.

一般のfに対して, f = f+ − f−と分解するとする. f+, f−に対して, 上記の議論を行えばよい. ◻

L∞の諸性質

以下, p = ∞の場合を扱う.

Definition 3.13. L∞(μ) ≔ ℒ∞(μ)/I. (ただし, ℒ∞(μ) ≔ {f : X → ℝ ∣ f は可測函数で, 本質的に有界.} I ≔ {fは本質的に有界な可測函数であり, f = 0 μ-a.e.}) $$\begin{aligned} \|f\|_{\infty} \coloneqq \inf \{ C \geq 0 \mid |f| \leq C \ \mu \text{-a.e.}\} \end{aligned}$$ これがノルム空間になることが明らかである.

Theorem 3.14. q = 1を p = ∞の共役とみなせば, ∀f ∈ L∞(μ), ∀g ∈ L1(μ)に対し, $$\begin{aligned} \int_X |fg| d\mu \leq \|f\|_{\infty}\|g\|_1 \end{aligned}$$

Proof. |f| ≤ ∥f∥∞ μ-a.e.より明らか. ◻

Theorem 3.15. L∞(μ)が完備である.

Proof. L∞-Cauchy列(fk)k ∈ ℕ を任意に取る. k, l ∈ ℕに対して, Ek, l ≔ {|fk| > ∥fk∥∞} ∪ {|fk−fl| > ∥fk − fl∥∞} と定める. よって, $$\begin{aligned} E \coloneqq \bigcup_{k,l \in \mathbb{N}} E_{k,l} \end{aligned}$$ とすれば, μ(E) = 0が成立する. X \ E上では, $$\begin{aligned} |f_k| &\leq \|f_k\|_{\infty}, \\ |f_k - f_l| &\leq \|f_k - f_l\|_{\infty}, \ \forall k, l \in \mathbb{N}. \end{aligned}$$ が成立し, そこで, $$\begin{aligned} f(x) \coloneqq \begin{cases} \lim_{k \to \infty} f_k(x) \ &(x \in X \setminus E) \\ 0 \ &(x \in E) \end{cases} \end{aligned}$$ と定めればよい. ◻

Remark 3.2. f, fk ∈ L∞(ℝ,ℬ(ℝ),λ), k ∈ ℕとして, fk(x) = e−|x|/k, f(x) = 1 とする. fkが fに各点収束する. ∥fk∥∞ = ∥f∥∞ = 1となるが, ∥fk − f∥∞ = 1. よって, 概収束かつノルム収束からL∞-収束を導くことができない.

Proposition 3.16. ℰ ∩ L∞(μ) = ℰ がL∞(μ)において稠密である.

Proof. f ∈ L∞(μ)を非負とする. この時, $$\begin{aligned} \tilde{f} (x)\coloneqq \begin{cases} f(x) \ &(x \in \{ f \leq \|f\|_{\infty}\} \\ 0 \ &(x \not\in \{f > \|f\|_{\infty}\} \end{cases} \end{aligned}$$ と定めるとf̃は有界で, $\|g_k - \tilde{f}\|_{\infty} \leq \frac{1}{2^k}$ を満たす非負単函数列が取れる. 一般のfに対して, f = f+ − f−と分解してそれぞれの近似列を考えれば良い. ◻

積測度とFubiniの定理

この節ではLebesgue積分論における極めて重要なFubiniの定理を紹介する.

測度の一意性

Definition 4.1 (π-system, λ-system). 集合Xに対して,

  1. 𝒢 ⊆ 𝒫(X)がπ-system $\xLongleftrightarrow{\mathrm{def}} \forall A,B \in \mathcal{G}, A \cap B \in \mathcal{G}$

  2. 𝒟 ⊆ 𝒫(X)が X 上のλ-system $\xLongleftrightarrow{\mathrm{def}}$ 以下の3つの性質を満たす:

    1. X ∈ 𝒟

    2. A, B ∈ 𝒟, A ⊆ B ⟹ B \ A ∈ 𝒟

    3. {Ak}k ∈ ℕ ⊆ 𝒟 が単調増大列  ⟹ ⋃k ∈ ℕAk ∈ 𝒟

Proposition 4.2. 𝒟 ⊆ 𝒫(X) をλ-systemとする. この時, $$\begin{aligned} \mathcal{D} \text{ is a $\pi$-system} \iff \mathcal{D} \text{ is a $\sigma$-algebra} \end{aligned}$$

Proof. "⟸"が明らかである. "⟹"を示す. X ∈ 𝒟 が明らかである. A ∈ 𝒟 に対し, X \ A ∈ 𝒟. 最後に可算和に対して閉じることを示す: {An}n ∈ ℕ ⊆ 𝒟 これを示すには, ∀A, B ∈ 𝒟; A ∪ B ∈ 𝒟 を示せば十分. A ∪ B = X \ ((X\A)∩(X\B)) より, 𝒟 が補集合と共通部分で閉じるので, ok. ◻

Proposition 4.3. π-system 𝒢 ⊆ 𝒫(X) とX上の λ-system 𝒟 に対して, 𝒢 ⊆ 𝒟 を満たせば, σ(𝒢) ⊆ 𝒟 が成立する.

Proof. より強いことを示す: δ(𝒢) ≔ ⋂{Λ ⊆ 𝒫(X) ∣ ΛはX上 𝒢 を含むλ-system} と定めると, 明らかにδ(𝒢)がλ-systemで, 以下これがσ-algebraであること を示せば良い. Proposition 4.2より, δ(𝒢)がπ-systemである ことを示せば十分. E ∈ δ(𝒢) に対し, DE ≔ {A ∈ δ(𝒢) ∣ A ∩ E ∈ δ(𝒢)} と する. ∀E ∈ δ(𝒢) に対し, DEが λ-systemであることを 示せば, ∀A, B ∈ δ(𝒢); A ∩ B ∈ δ(𝒢)が従う. 以下, このことを確かめる. X ∩ E = E ∈ δ(𝒢)より, X ∈ DE. ∀A ⊆ B ∈ DEに対し, (B\A) ∩ E = (B∩E) \ (A∩E) ∈ δ(𝒢)より, B \ A ∈ DE が従う. 最後に, DE上の単調列 {An}n ∈ ℕ に対し, (⋃n ∈ ℕAn) ∩ E = ⋃n ∈ ℕ(An∩E) ∈ δ(𝒢) (∵ {An ∩ E} がδ(𝒢) 上の単調列) より, ⋃n ∈ ℕAn ∈ DEが成立する. 以上より, DEが X上の λ-systemである. ◻

Theorem 4.4. 測度空間(X,𝒮,μ) と𝒢 ⊆ 𝒫(X)は以下の条件を満たすとする:

  1. 𝒮 = σ(𝒢)

  2. 𝒢 はπ-systemで, ∃{Gn}n ∈ ℕ ⊆ 𝒢 s.t. {Gn}n ∈ ℕ が単調増大列かつ⋃n ∈ ℕGn = X

  3. ∀n ∈ ℕ; μ(Gn) < ∞

この時, 別の測度μ′に対して, ∀G ∈ 𝒢 μ(G) = μ′(G) を満たせば, μ = μ′が成立する.

Proof. 各n ∈ ℕに対して, $$\begin{aligned} \mathcal{D}_n \coloneqq \{ A \in \mathcal{S} \mid \mu(A\cap G_n) = \mu'(A_n \cap G_n)\} \end{aligned}$$ とおく. 以下, 𝒢 ⊆ 𝒟n, 𝒟nはX上の λ-systemが成立すると仮定すると, σ(𝒢) ⊆ 𝒟nが成立する. つまり, ∀A ∈ 𝒮, ∀n ∈ ℕ に対して, $$\begin{aligned} \mu(A\cap G_n) = \mu'(A\cap G_n) \end{aligned}$$ が成立する. 測度の連続性より, μ(A) = μ′(A)が得られる.

以下 𝒢 ⊆ 𝒟nと𝒟nがX上の λ-systemであることを示す:

G ∈ 𝒢 とすると, G ∩ Gn ∈ 𝒢 が成立するので, μ(G∩Gn) = μ′(G∩Gn)が得られ, G ∈ 𝒟nが従う.

また, μ(X∩Gn) = μ′(X∩Gn)が成立するので, X ∈ 𝒟nが成立. A, B ∈ 𝒟n, A ⊆ Bとすると, $$\begin{aligned} \mu\left( (B \setminus A) \cap G_n \right) &= \mu (B \cap G_n) - \mu(A \cap G_n) \\ &= \mu'(B\cap G_n) - \mu(A\cap G_n) \\ &= \mu'\left( (B\setminus A) \cap G_n \right) \end{aligned}$$ が成立し, B \ A ∈ 𝒟nが従う. さらに, {Ak}k ∈ ℕ ⊆ 𝒟n を単調増大列とすると, 測度の連続性より, $$\begin{aligned} \mu \left( \left( \bigcup_{k \in \mathbb{N}} A_k \right) \cap G_n \right) &= \lim_{k \to \infty} \mu(A_k \cap G_n) \\ &= \lim_{k \to \infty} \mu'(A_k \cap G_n) \\ &= \mu'\left( \left( \bigcup_{k \in \mathbb{N}} A_k \right) \cap G_n \right) \end{aligned}$$ が成立し, ⋃k ∈ ℕAk ∈ 𝒟nが成立する. ◻

Remark 4.1. 上記の集合列{Gn}n ∈ ℕ ⊆ 𝒫(X) をXの exhausting sequence と呼んで, Gn ↗ X と記す.

Definition 4.5. 測度空間 (X,𝒮,μ) に対して, 単調増大列 {Gn}n ∈ ℕ ⊆ 𝒮 s.t. X = ⋃n ∈ ℕGn を満たすものをXの 取り尽くし列と呼び, 各 n ∈ ℕに対して, μ(Gn) < ∞を満たすものを 有限測度の取り尽くし列という.

直積測度

測度空間(X,𝒮,μ), (Y,𝒯,ν) を固定する.

Definition 4.6. 𝒮 × 𝒯 ≔ {A × B ∣ A ∈ 𝒮, B ∈ 𝒯} をrectangular set という.

𝒮 ⊗ 𝒯 ≔ σ(𝒮 × 𝒯) を直積σ-加法族と呼ぶ. (X×Y,𝒮⊗𝒯) を直積可測空間と呼ぶ.

Definition 4.7. 測度空間(X,𝒮,μ)がσ-有限であるとは Xの有限測度の取り尽くし列が存在する時をいう.

Proposition 4.8. σ-有限な測度空間の直積で, rectangular setの測度をものの測度の積となる直積測度が一意に存在する.

Proof. 𝒢 と置くと, 𝒢 がπ-systemである. X, Yの有限測度の取り尽くし列をそれぞれ {An}n ∈ ℕ, {Bn}n ∈ ℕ とする. 各n ∈ ℕに対し, $$\begin{aligned} G_n \coloneqq A_n \times B_n \end{aligned}$$ と置くと, Gn ↗ X × Yとなる. ここでTheorem 4.4より一意性が従う. ◻

以下, X, Yの取り尽くし集合を {An}n ∈ ℕ, {Bn}n ∈ ℕ と, Gn ≔ An × Bnを固定する.

Proposition 4.9. ∀E ∈ 𝒮 ⊗ 𝒯 に対して, 次が成立する:
∀y ∈ Yに対して, X ∋ x ↦ 𝟙E(x,y) は𝒮-可測,
∀x ∈ Xに対して, Y ∋ y ↦ 𝟙E(x,y) は𝒯-可測,
X ∋ x ↦ ∫Y𝟙E(x,y)dν(y)は𝒮-可測,
Y ∋ x ↦ ∫X𝟙E(x,y)dμ(x)は𝒯-可測,
$$\begin{aligned} \int_Y \left(\int_X \mathds{1}_E(x,y)d \mu(x)\right) d\nu (y) = \int_X \left( \int_Y \mathds{1}_E(x,y) d\nu (y) \right) d\mu(x) \end{aligned}$$

Proof. 各n ∈ ℕに対して, $$\begin{aligned} \mathcal{D}_n \coloneqq \{D \in \mathcal{S}\otimes \mathcal{T} \mid E = D \cap G_n \text{として示したい条件を満たす. }\} \end{aligned}$$ 𝒮 × 𝒯 ⊆ 𝒟n, 𝒟nはX × Y上のλ-systemが分かれば, $$\begin{aligned} \sigma(\mathcal{S}\times \mathcal{T}) \subseteq \mathcal{D}_n \end{aligned}$$ が成立する. つまり, 任意のE ∈ 𝒮 ⊗ 𝒯 に対し, $$\begin{aligned} E_n \coloneqq E \cap G_n \end{aligned}$$ は示したい条件を満たす. すると, ∀y ∈ Yに対して, $$\begin{aligned} X \ni x \mapsto \mathds{1}_E(x,y) = \lim_{n \to \infty} \mathds{1}_{E_n}(x,y) \end{aligned}$$ が成立するので, 極限が可測性を保つので, 最初の主張が示される. 同様に2番目の主張も成立する. 次に, 単調収束定理より $$\begin{aligned} X \in x \mapsto \int_Y \mathds{1}_E(x,y) d \nu (y) = \lim_{n \to \infty} \int_Y \mathds{1}_{E_n}(x,y) d\nu (y) \end{aligned}$$ が成立し, この函数が𝒮-可測である. 3番目の主張が従う. 同様に4番目の主張も成立する. 単調収束定理より, 極限と積分の順序の入れ替えを行えば, 最後の主張が従う.

これから𝒮 × 𝒯 ⊆ 𝒟nと𝒟nがX × Y上のλ-systemであることを示す: ∀A × B ∈ 𝒮 × 𝒯 に対し, (A×B) ∩ Gn = (A∩An) × (B∩Bn) となる. よって, E = (A×B) ∩ Gnとして, $$\begin{aligned} \mathds{1}_E(x,y) = \mathds{1}_{A\cap A_n}(x) \mathds{1}_{B\cap B_n}(y) \end{aligned}$$ となる. これより, E ∈ Dnが明らかである.

最後に, 𝒟nがX × Y上のλ-systemであることを示す: D, D′ ∈ 𝒟n, D ⊆ D′とする. この時, $$\begin{aligned} \mathds{1}_{(D'\setminus D) \cap G_n} = \mathds{1}_{D' \cap G_n} - \mathds{1}_{D\cap G_n} \end{aligned}$$ が成立するので, D′ \ D ∈ 𝒟n が簡単に確認できる. 増大列に関する議論は極限は可測性を保つことと 単調収束定理より直ちに従う. ◻

上記の議論の帰結として, 次の定理が成立する.

Theorem 4.10. m : 𝒮 ⊗ 𝒯 → [0,∞]を, $$\begin{aligned} m(E) &\coloneqq \int_Y\left( \int_X \mathds{1}_E(x,y) d\mu (x) \right) d\nu (y) \\ &= \int_X \left( \int_Y \mathds{1}_E (x,y) d\nu (y) \right) d\mu (x) \end{aligned}$$ が矩形集合を測度の積に対応させる(X×Y,𝒮⊗𝒯) 上の唯一の測度.

Proof. 空集合の定義函数が恒等的に0である. 次に, {Ek}k ∈ ℕ ⊆ 𝒮 ⊗ 𝒯 を互いに素とする. それぞれの集合の定義函数に対して単調収束定理を適用すればよい. ◻

Definition 4.11. Theorem 4.10で定めたmを μ × νの直積測度と呼び, μ × ν と記す. (X×Y,𝒮⊗𝒯,μ×ν) を直積空間と呼ぶ.

Fubiniの定理

以降, (X,𝒮,μ), (Y,𝒯,ν)をσ-有限な測度空間とする.

Theorem 4.12. 非負で𝒮 ⊗ 𝒯-可測な∀f : X × Y → [0,∞]に対し, 次が成立する: ∀y ∈ Yに対して, X ∋ x ↦ f(x,y) は𝒮-可測,
∀x ∈ Xに対して, Y ∋ y ↦ f(x,y) は𝒯-可測,
X ∋ x ↦ ∫Yf(x,y)dν(y)は𝒮-可測,
Y ∋ x ↦ ∫Xf(x,y)dμ(x)は𝒯-可測,
$$\begin{aligned} \int_{X \times Y} f d (\mu \times \nu ) = \int_Y \left(\int_X f(x,y)d \mu(x)\right) d\nu (y) = \int_X \left( \int_Y f(x,y) d\nu (y) \right) d\mu(x) \end{aligned}$$

Proof. 非負可測関数の単函数による近似列を用いて, 単調収束定理とProposition 4.9を適用すれば良い. ◻

Theorem 4.13.

$f : X \times Y \to \overline{\mathbb{R}}$ を𝒮 ⊗ 𝒯-可測で,

$$\begin{aligned} & \int_{X \times Y}|f| d \left( \mu \times \nu \right), \\ \int_Y\left( \int_X |f(x,y) | d \mu (x) \right)& d\nu (y), \quad \int_X \left( \int_Y |f(x,y)| d\nu (y) \right) d \mu (x) \end{aligned}$$ の少なくとも1つが有限となるものとする. この時, f ∈ L1(μ×ν)であって,
ν-a.a. y ∈ Yに対し, f(⋅,y) ∈ L1(μ),
μ-a.a. x ∈ Xに対し, f(x,⋅) ∈ L1(ν),
Y ∋ y ↦ ∫Xf(x,y)dμ(x)はL1(ν)の元,
X ∋ x ↦ ∫Yf(x,y)dν(x)はL1(μ)の元, $$\begin{aligned} \int_{X\times Y}f d(\mu \times \nu ) = \int_Y\left( \int_X f(x,y) d\mu (x) \right) d\nu (y) = \int_X \left( \int_Y f(x,y) d\nu (y) \right) d\mu (x) \end{aligned}$$ が成立する.

Proof. |f|は非負の可測函数であり, Theorem $\ref{theorem:Tonelli}$から, 1つが有限ならば, ∫X × Y|f|d(μ×ν) < ∞ が得られる. 可積分性より, ν-a.a. y ∈ Yに対し, f(⋅,y) ∈ L1(μ)と μ-a.a. x ∈ Xに対し, f(x,⋅) ∈ L1(ν) が従う. また, f± ≤ |f| より, $$\begin{aligned} \int_X f^{\pm} (x,y) d \mu (x) \leq \int_X |f(x,y)| d\mu (x), \ \forall y \in Y \end{aligned}$$ がyについて可積分なので, $$\begin{aligned} Y \ni y \mapsto \int_X f^{\pm} (x,y) d\mu (x) \end{aligned}$$ はともに𝒯-可測で, L1(ν)の元, よって, Y ∋ y ↦ ∫Xf(x,y)dμ(x)もL1(ν)の元である.

Proposition 4.14. Theorem 5.17 (Riemann-Lebesgueの補題). f ∈ L1(λ)とt ∈ ℝに対し, x ↦ f(x)cos (tx), f(x)sin (tx)はL1(λ)の元であることに注意して, ∀f ∈ L1(λ)に対し, $$\begin{aligned} \lim_{|t| \to \infty} \int_{\mathbb{R}} f(x) \cos(tx) d\lambda(x) = 0 \end{aligned}$$ が成立, cos をsin にしても良い.

Proof. ∀ε > 0を固定. Theorem 5.16 を p = 1として用いて, ∃φε ∈ Cc∞(ℝ) s.t. $$\begin{aligned} \int_{\mathbb{R}} |f-\varphi_{\varepsilon}|d\lambda < \varepsilon \end{aligned}$$ そこで, suppφε ⊆ [−R,R] となるRを取り, ∀t ≠ 0に対し, $$\begin{aligned} \int_{\mathbb{R}}\varphi_{\varepsilon}(x) \cos(tx) d \lambda(x) & = - \frac{1}{t} \int_{\mathbb{R}}(\partial_x\varphi_{\varepsilon})(x) \sin(tx) d\lambda(x) \end{aligned}$$ よって, $$\begin{aligned} \left| \int_{\mathbb{R}}\varphi_{\varepsilon}(x) \cos(tx) d\lambda(x) \right| \leq \frac{1}{t} \int_{\mathbb{R}}|\partial_{x}\varphi_{\varepsilon}| d\lambda \end{aligned}$$ すると, It ≔ ∫ℝf(x)cos (tx)dλ(x) とおいて, $$\begin{aligned} |I_t| &= \left| \int_{\mathbb{R}}\left( f(x) - \varphi_{\varepsilon}(x) + \varphi_{\varepsilon}(x)\right) \cos(tx) d\lambda(x) \right| \\ &\leq \int_{\mathbb{R}}\left| (f(x) - \varphi_{\varepsilon}(x) \cos(tx) d\lambda(x) \right| + \left| \int_{\mathbb{R}}\varphi_{\varepsilon}(x) \cos(tx) d\lambda(x) \right| \\ &< \varepsilon + \frac{1}{|t|} \int_{\mathbb{R}} |\partial_x\varphi_{\varepsilon}| d\lambda . \end{aligned}$$ これより, $$\begin{aligned} \limsup_{|t| \to \infty} |I_t| \leq \varepsilon. \end{aligned}$$ εが任意より, 定理が従う. ◻

Radon-Nikodymの定理

以下, 測度空間(X,𝒮,μ)を測度空間として固定する.

Definition 6.1. (X,𝒮) 上の測度 νがμに関して絶対連続である$\xLongleftrightarrow{\mathrm{def}} \forall N \in \mathcal{S}$ s.t. μ(N) = 0 ⟹ ν(N) = 0. これを ν ≪ μと表す.

Example 6.2. f : X → [0,∞]を𝒮-可測函数とする. $$\begin{aligned} \nu (A) \coloneqq \int_A f d\mu , \quad A \in \mathcal{S} \end{aligned}$$ νが(X,𝒮) 上の測度である. (∵単調収束定理) 更に, ν ≪ μ.

Theorem 6.3. μ, νをσ-有限な(X,𝒮)上の測度で, ν ≪ μとする. この時, ある非負可測函数f : X → [0,∞]が存在して, $$\begin{aligned} \nu (A) = \int_A f d\mu , \quad \forall A \in \mathcal{S} \end{aligned}$$ fは μに関して一意に存在する.

Proof. まず, μ, νが有限測度の時:

証明1: (これは面倒くさいし, 発想も自然ではないし, これで何も勉強できない. だけど, 実数の完備性さえ知っていれば証明として読めるやつになっている)

Step 1. ρ ≔ μ + νに対し, $$\begin{aligned} \Phi(u)\coloneqq \int_X u^2 d\mu + \int_X (1-u)^2 d\nu \end{aligned}$$ すると, Φ(0) = ν(X) < ∞より, $$\begin{aligned} C\coloneqq \inf \{\Phi(u) \mid u \in L^2(\rho)\} \end{aligned}$$ は有限値. また, ∀u, v ∈ L2(ρ)に対し, $$\label{eq:6.5} \begin{aligned} \Phi(u+v) + \Phi(u-v) = 2 \Phi(u) + 2 \|v\|^2_{L^2(\rho)} \end{aligned}$$ は計算より得られる.

Step 2. ∃g ∈ L2(ρ) s.t. C = Φ(g). 以下これを示す: inf の定義より, ∃{gn}n ∈ ℕ s.t. C = limn → ∞Φ(gn). すると, $$\begin{aligned} u \coloneqq \frac{g_n+g_m}{2}, \quad v \coloneqq \frac{g_n -g_m}{2} \end{aligned}$$ と取って, ([eq:6.5])式に代入すると, $$\begin{aligned} C \xlongleftarrow{n,m \to \infty} & \frac{1}{2}\Phi(g_n) + \frac{1}{2}\Phi(g_m) \\ &= \Phi\left( \frac{g_n+g_m}{2} \right) + \left\| \frac{g_n -g_m}{2} \right\|^2_{L^2(\rho)} \geq C + \left\| \frac{g_n -g_m}{2} \right\|^2_{L^2(\rho)} . \end{aligned}$$ よって, {gn}n ∈ ℕ がL2(ρ)上のCauchy列である. それの極限を gと定める. 最後に, u = gn, v = g − gnとして, ([eq:6.5])式に代入して, $$\begin{aligned} &\Phi(g) + \Phi(2g_n-g) \\ &=2\Phi(g_n) + 2\|g-g_n\|^2_{L^2(\rho)}\\ &\xlongrightarrow{n\to \infty} 2C+0 \end{aligned}$$ これに従って, Φ(g) ≤ Cが得られ, Φ(g) = Cが得られる.

Step 3. gは g : X → [0,1]とできる. $$\begin{aligned} \tilde{g} \coloneqq \min \{g^+ ,1\} \end{aligned}$$ と置くと, 1 − g̃ = (1−g+)+より, $$\begin{aligned} \left( \tilde{g} \right) ^2 & \leq g^2 \\ (1-\tilde{g})^2 &\leq (1-g)^2 \end{aligned}$$ よって, $$\begin{aligned} C \leq \Phi(\tilde{g}) \leq \Phi(g) =C \end{aligned}$$ そこで, g̃を gとして使えば良い.

Step 4. $$\begin{aligned} f\coloneqq \frac{g}{1-g}\mathds{1}_{g<1} \end{aligned}$$ が定理の主張を満たすことを示せば良い. ∀A ∈ 𝒮, ∀t ∈ ℝに対し, $$\begin{aligned} 0 &\leq \Phi(g+ t \mathds{1}_A) - \Phi(g) \\ &= t^2\left( \mu (A) + \nu (A) \right) + 2t \left( \int_A g d\mu - \int_A (1-g) d\nu \right) \end{aligned}$$ が成立するので, 2次函数の判別式より, $$\begin{aligned} \int_{A}g d \mu = \int_A (1-g) d\nu , \quad \forall A \in \mathcal{S} \end{aligned}$$ そこで, A = {g = 1} と取ると, μ({g=1}) = 0, いま, ν ≪ μより, $$\begin{aligned} \nu \left( \{g=1\} \right) =0 \end{aligned}$$ が成立する. また非負可測函数hを単函数を持って単調増加的に近似できるので, $$\begin{aligned} \int_A gh d\mu = \int_A (1-g)h d\nu , \quad \forall A \in \mathcal{S}. \end{aligned}$$ よって, $h= \frac{1}{1-g}\mathds{1}_{\{g<1\} }$ と取ると, $$\begin{aligned} \int_A \frac{g}{1-g}\mathds{1}_{\{g<1\} }d\mu = \int_A \mathds{1}_{\{g<1\}} = \nu (A). \end{aligned}$$

証明2:(少し函数解析の知識を使う)

L2(μ+ν)が前のchapterで完備距離空間であることがわかる. さらに, これに自然な 内積構造が入る. i.e. ∀f, g ∈ L2(μ+ν), ⟨f, g⟩ ≔ ∫Xfgd(μ+ν).

これにより, L2(μ+ν)がHilbert空間であることがわかった. 今, この空間上の線形汎函数 φ : L2(μ+ν) → ℝを f → ∫Xfdν として定める. Hilbert 空間のRieszの表現定理より, あるg ∈ L2(μ+ν)が存在して, ∀f ∈ L2(μ+ν)に対して, $$\begin{aligned} \int_X f d \nu = \int_X fg d (\mu + \nu) = \int_X fg d \mu + \int_X fg d \nu \end{aligned}$$ が成立する. この式を変形すると, $$\label{eq:radon} \begin{aligned} \int_X f (1-g) d \nu = \int_X fg d\mu \end{aligned}$$ が成立する.

f ≔ 𝟙{g < 0} とすると, 上式より0 ≤ ν({g<0}) ≤ ∫Xfgdμ ≤ 0. 従って, μ({g<0}) = 0. (ここに多少ギャップを感じるかもしれないが, よく考えて見たら 当たり前のことしか言っていない, checkせよ) f ≔ 𝟙{g ≥ 1} とすると, 同様に, μ({g≥1}) = 0 が得られる.

ν ≪ μより, ν({g<0}) = ν({g≥1}) = 0 がわかる.

任意の可測集合A ∈ 𝒮 に対して, $f_k\coloneqq \min \{ \frac{\mathds{1}_{A }}{1-g} , k\}$ と定めると, 式[eq:radon] に対して, 単調収束定理を 適用する.

$$\begin{aligned} \int_X \mathds{1}_A d \nu = \int_{X} \mathds{1}_A \frac{g}{1-g} d\mu \end{aligned}$$

が成立する. このg/(1−g)が我々の望む函数である.

以上より存在性の証明が終わった. 一意性の証明を読者の演習問題にする. σ-有限な測度空間の場合の議論も読者に任せる. ◻
